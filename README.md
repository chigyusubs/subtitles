## Usage

Download the `.vtt` file for your episode and load it in your media player:

- **mpv**: `mpv video.mp4 --sub-file=s01e01.en.vtt`
- **VLC**: Subtitles → Add Subtitle File
- **MPC-HC / MPC-BE**: File → Load Subtitle
